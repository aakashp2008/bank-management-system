# 🏦 Bank Management System

## 📌 Project Overview

The **Bank Management System** is a Python-based application that simulates basic banking operations through a menu-driven interface. It enables users to perform common banking tasks such as checking their account balance, depositing money, and withdrawing funds while ensuring valid transaction handling.

---

## ✨ Features

- Check account balance
- Deposit money into the account
- Withdraw money with balance verification
- Input validation for secure transactions
- Menu-driven interface for easy navigation
- Exit the application safely
