# Bank Management System

## 🏦 About the Project

The Bank Management System is a simple Python application that simulates basic banking operations through a menu-driven interface.

## ✨ Features

* Check account balance
* Deposit money
* Withdraw money
* Input validation for transactions
* Exit the application

## 🛠️ Technologies Used

* Python

## 🎯 Learning Outcomes

This project helped me learn:

* Variables and arithmetic operations
* Loops and conditional statements
* Menu-driven programming
* Basic banking transaction logic
* Input validation
